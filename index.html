    Review: Flexbox

  • Volver al inicio

    • display: flex

    cambia un elemento a un contenedor de nivel de bloque con elementos flexibles dentro de él.

    display: inline-flex

    permite que varios contenedores flexibles aparezcan alineados entre sí.

    justify-content

    se utiliza para espaciar elementos a lo largo del eje principal.

    align-items

    se utiliza para espaciar elementos a lo largo del eje transversal.

    flex-grow

    se utiliza para especificar cuánto espacio (y en qué proporciones) absorben los elementos flexibles a lo largo del eje principal.

    flex-shrink

    se utiliza para especificar cuánto se encogen los elementos flexibles y en qué proporciones a lo largo del eje principal.

    flex-basis

    se utiliza para especificar el tamaño inicial de un elemento con estilo flex-growy/o flex-shrink.

    flex

    se utiliza para especificar flex-grow, flex-shrinky flex-basisen una declaración.

    flex-wrap

    especifica que los elementos deben desplazarse a lo largo del eje transversal si el contenedor flexible no es lo suficientemente grande.

    align-content

    se utiliza para espaciar filas a lo largo del eje transversal.

    flex-direction

    se utiliza para especificar los ejes principal y transversal.

    flex-flow

    se utiliza para especificar flex-wrapy flex-directionen una declaración.

    Los contenedores flexibles

    se pueden anidar unos dentro de otros declarando display: flexo display: inline-flexpara hijos de contenedores flexibles.
